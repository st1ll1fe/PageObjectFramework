1. common package
Пакет включает в себя общие классы, по которым будет работать в дальнейшем фреймворк

1.1 CommonActions
Создание экземпляра WebDriver
Открытие Вебдрайвера на весь экран
Установка неявного ожидания

1.2 Config
Файл содержит константы, конфигурирующие проект

2. constants package
Пакет включает в себя классы с группировкой по константам

2.1 Constant
Константы явного и неявного ожидания, константа с url

3. pages package
включает в себя подпакеты с страницами

3.1 base package
BasePage Базовый класс, который будет наследоваться остальными
Прописана логикика открытия странички
Логика для явных ожиданий видимости элемента

3.2 cian_home package
CianHomePage Класс, отвечающий за домашнюю страницу сайта https://spb.cian.ru/
Наследует BasePage
включает в себя описание локаторов, описание методоа на странице

3.3 listing package
CianListingPage Отсортированная страница



ТЕСТЫ

1. test.java.tests.base
BaseTest Базовый класс, который будет наследоваться остальными
